# CLAUDE.md — プロジェクト引き継ぎメモ

このファイルは Claude Code が最初に読む、プロジェクトの背景説明です。

## これは何
株式会社京伸(大阪・精密板金加工)の、Instagram プロフィールに貼る
「リンクまとめページ(リットリンク代替)」。単一の `index.html` で完結する
静的サイト。ビルド工程・依存パッケージなし。ブラウザで開けば動く。

公式サイト: https://www.f-kyo-shin.co.jp/

## 目的
Instagram の自己紹介文は文字数制限があるため、このページ1枚で
会社の良さ(若いチーム文化・スピードと精度・最新レーザー設備・実績)が
すべて伝わるようにする。インスタ → このページ → 各詳細ページ/SNS へ誘導。

## デザイン方針(重要)
- コーポレートカラー: 赤・黒・白。赤は発光レッド #ff2630(ロゴ由来は #e60b10)
- 現在はダークテーマ(ガンメタル背景)。レーザー切断をモチーフにした
  アンビエント背景(赤い走査線＋火花パーティクル)が signature 要素
- ロゴは画像ではなく SVG で再現(六角形マーク＝赤3枚＋白/黒3枚の三角形、
  Kyo=赤・shin=白、ハイフンは赤黒の斜め分割)。原本のロゴ画像があれば差し替え可
- ターゲットは若い世代(求職者含む)。"今どき"でカッコいい見た目を重視
- 文字が読みにくくならない範囲でアニメーションを使う。prefers-reduced-motion 対応済み

## 構成セクション(index.html 内)
1. ヒーロー(ロゴ＋キャッチ＋工場動画＋ステッカー)
2. Culture「京伸ってこんな会社」(20代主力 / 役職者も若い / 仲が良い)
3. Links(見積もり・公式HP・採用・動画)
4. Works(製作実績スライドショー)
5. Business(事業内容を横スワイプ)
6. Pick up(最新設備＋K-MAXフレキシブルパワーアーム)
7. Spec(対応素材・データ・所在地)
8. Follow us(Instagram / TikTok / YouTube / Facebook / X)
9. 下部固定CTA(電話する / 見積もり依頼)
